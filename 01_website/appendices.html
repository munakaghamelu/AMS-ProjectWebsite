Deployment Manual

  • Set up Kinects as in ideal room plan at specific known distances, with each Kinect connected to it's own computer, where each computer needs a copy of our Unity application.

  • Navigate to the recording scene in the unity project. Before recording, select which Kinect number your computer is in the set up. Kinect 1 should be the main Kinect, with the number increasing going anti-clockwise 90 degrees.

  • Once you have the correct Kinect number selected on each machine, make sure that each Kinect is tracking the dance subject successfully before beginning to record. Then, begin recording on each machine as simultaneously as possible.

  • Stop the recording when desired on each machine. Name each file and upload every desired file to the cloud.

  • On one of the machines, navigate to the watch/analyse panel. Load up the list of available files in the cloud and download the files pertaining to the dance that was just performed.

  • Click merge files, and select 1-4 dance files (from the same dance!) to be merged

  • Click select dance, and choose the merged dance file (NB: Currently the merged file will be saved as test.txt). The watch playback button should now be active, clicking it will lead to the screen that will allow you to watch the playback of the merged dance in full.

  • Begin playback, experiment with focusing on specific parts of the body, rotating the playback with the arrow keys, navigating the dance using the time bar, and leaving notes about the dance.
  • You can also select the Compare tab in the watch/analyse scene. This allows you to choose two dances to compare. This will give you a score /100 as well as a breakdown on how you performed in different areas.
